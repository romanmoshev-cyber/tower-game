## 💡 Ключевые улучшения

### Было:
```css
/* Проблемы на мобилах */
height: 100vh; /* Скачки при скрытии адресной строки */
touch-action: none; /* Блокировка скроллинга */
```

### Стало:
```css
/* Оптимизировано для мобилок */
height: 100svh; /* Стабильная высота */
padding: env(safe-area-inset-*); /* Безопасные отступы */
touch-action: manipulation; /* Нормальный скроллинг */
-webkit-overflow-scrolling: touch; /* Плавная прокрутка iOS */
```
